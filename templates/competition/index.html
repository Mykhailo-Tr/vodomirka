Competition Mode Help
Getting Started
  1. Create Exercise: Define shooting exercises or use system defaults (ГП-11, ГП-11a, ГП-12)
  2. Create Competition: Select exercise and participating athletes
  3. Start Competition: Begin the competition when ready
  4. Manage Series: Athletes shoot in any order, series can be finished early
  5. Finish Competition: Complete when all series are finished or finished early
Key Features
  • Multi-athlete competitions with flexible shooting order
  • Series-based shooting with partial completion support
  • Early series finishing (doesn't block competition completion)
  • Image upload from file or camera
  • Real-time scoring and statistics
  • Competition resume capability
