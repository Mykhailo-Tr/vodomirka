Target Scorer

Інтелектуальна система автоматизованого аналізу паперових та картонних мішеней для спортивної стрільби на основі комп’ютерного зору та штучного інтелекту (ISSF 10 м, пневматична зброя).

YOLOv11 Computer Vision ISSF Best Edge Python + Flask Roboflow

Призначення системи

Target Scorer призначена для автоматичного визначення кількості пострілів, їх координат та результатів відповідно до правил ISSF з використанням звичайних камер та недорогого обладнання. Система орієнтована на навчально-тренувальний процес у стрілецьких секціях, школах та клубах.

Ключові можливості

Комп’ютерний зір

Детекція отворів від куль, центру мішені та залікових кілець за допомогою моделі YOLOv11 Object Detection.

Точний підрахунок

Результат пострілу визначається за принципом ISSF best edge з переведенням координат з пікселів у міліметри.

Вебінтерфейс

Візуалізація результатів стрільби, накладання мішені, збереження та аналіз статистики у зручному форматі.

Технологічний стек

  • Python — серверна логіка та обчислення
  • OpenCV, NumPy — обробка зображень та геометричні розрахунки
  • Roboflow + YOLOv11 — навчання та інференс моделі ШІ
  • Flask — вебсервер та API
  • HTML / CSS / JavaScript — клієнтський інтерфейс

Результати тестування

Система протестована на 100 мішенях (176 пострілів). Коректне визначення кількості пострілів — 97%. Відхилення результату не перевищувало ±1 балу.

Перспективи розвитку

  • Підтримка рухомих мішеней та інших дисциплін ISSF
  • Підвищення точності при щільних групах пострілів
  • Розширена аналітика та профілі спортсменів
